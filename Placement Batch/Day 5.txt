#include <stdio.h>

int main()
{
    int a=10,b,c;
    b=c++;
    c=--a;
    a=++b;
    printf("%d\t%d\t%d",a,b,c);                                 OUTPUT
    return 0;                                                                      1   1     9
}
-------------------------------------------------------------------------
NOTES :
*Turbo C: For Solving Multiple Unary operators at a Time.Compiler will solve first \\
  ----------
The pre-operations(pre-increment or pre-Decrement) Than it put the value at 
place of variable or solve it  and at last we will perform the post increment.
 Special COnditions: 
 ----------------------   
*If  the last value is the post (INcrement or Decrement)
 AND The output of the whole opreations will be assingned into the value?
  --------------------------------------------------------------------------------------
then we will not perform the last operation into the output.

*If we perform the unary operations into the printf statement?
   -------------------------------------------------------------------------
 then compiler will start checking the operation from right to left.

*Online Compiler: we will start solving  from left to Right. 
   -------------------
