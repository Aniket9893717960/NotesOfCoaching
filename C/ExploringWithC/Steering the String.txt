NOTES:
The ! (logical NOT) operator has higher precedence than >= (greater than or equal to) operator.
 As a result, the expression !a >= 400 is equivalent to (!a) >= 400.
 The logical NOT of a (500) is 0 (false), so the condition becomes 0 >= 400, which is false.
----------------------------------------------------------
#include <stdio.h>
int main()
{
    int a=500,b=100,c;
    if(!a>=400)
    b=300;                                                                         OUTPUT
    c=200;                                                                         b=100 c=200
    printf("b=%d c=%d",b,c);
    return 0;
}

8)
----------------------------------------------------------
NOTES:
The logical NOT operator (!) is used here. It flips the truth value of its operand.
 In C, any non-zero value is considered true, and zero is considered false. 
So, !x will be 0 (false) since x is initially 10 (true), and !y will be 0 (false) since y is initially -20 (true).
----------------------------------------------------------
#include <stdio.h>
int main()
{
   int x=10,y=-20;
   x=!x;                                                                         OUTPUT
   y=!y;                                                                         x=1  y=1
   printf("x=%d y=%d\n",x,y);
   return 0;
}

21)
--------------------------------------------------
Notes:
 This Line         a = c > 1 ? d > 1 || e > 1 ? 100 : 200 : 300;       is a nested Conditional expression.
                          ------------------------------------------------
The outer conditional checks if c is greater than 1. If true, it evaluates the inner conditional;
 otherwise, it assigns the value 300 to a.
The inner conditional checks if d > 1 || e > 1 is true. If true, it assigns 100 to a; otherwise, it assigns 200.
----------------------------------------------------
#include <stdio.h>
int main()
{
   int c=0,d=5,e=10,a;
   a=c>1?d>1||e>1?100:200:300;
   printf("a = %d\n",a);
   return 0;
}
